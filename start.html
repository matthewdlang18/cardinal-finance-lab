üöÄ Stock Analysis Lab Setup

The Stock Analysis Lab requires a backend server to securely fetch data from Polygon.io. Follow these steps to get started:

Step 1: Install Dependencies

Open Terminal and navigate to the personal-finance folder, then run: 

cd /Users/mattlang/Dropbox/Matt/2026/IFDM/personal-finance
npm install

Step 2: Start the Server

Start the backend server: 

npm start

You should see: Server running on http://localhost:3001

Step 3: Verify Connection

Open test-api.html and click "Test Server Connection"

Step 4: Start Analyzing

Once the server is running, open stock-analysis.html and start analyzing stocks!

üìù Note about API Key Your Polygon.io API key is stored in .env and is already configured. The free tier allows 5 requests per minute, which is sufficient for educational use.

Troubleshooting

  • 401 Error: Check that your API key in .env is correct
  • Connection Refused: Make sure the server is running (npm start)
  • Rate Limit: Wait 1 minute between requests (free tier limit)
  • No Data: Try a different date range or ticker symbol
